<h1>Jaxon's Jewelry</h1>
<p>A dynamic jewelry website built using PHP, HTML, CSS and MySQL as part of a web development course project. This site showcases jewelry collections and includes interactive features like a contact form and a birthstone challenge.</p>
<br>
<h2>Project Overview</h2>
<p>The project focuses on:</p>
<ul>
  <li>Server-side scripting with PHP</li>
  <li>Form validation and sanitization</li>
  <li>Database integration using MySQL</li>
  <li>Reusable layouts with PHP includes</li>
  <li>Basic responsive design and image optimization</li>
  
</ul>
