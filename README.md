Jaxon’s Jewelry
A multi-page website designed to simulate a real world jewelry business. The project focuses on:

Server-side scripting with PHP
Form validation and sanitization
Database integration using MySQL
Reusable layouts with PHP includes
Basic responsive design and image optimization

Technologies Used:
HTML5 – Page structure
CSS3 – Styling and layout
PHP – Server-side functionality
MySQL – Database for storing user data
phpMyAdmin – Database management
MAMP – Local development environment
